## What actually shipped

The big one: every Claude Code skill we use daily at Benmore is now public at
[/skills](/skills) with a `/raw` endpoint per skill so any LLM can pull the full
SKILL.md as plain text. 71 skills across payments, compliance, design eng, AI SEO,
CLI, frontend, and tooling.

Cattle Logic crossed a real threshold this week — first in-person ranch
walkthrough in Amarillo. The onboarding flow is the kind of thing that only gets
better when you watch someone with cracked, sun-cracked hands try to use it on
a phone in 102°F heat. Three changes shipped the next day.

## Reading

Andreessen's piece on agentic infra reframes "AI infrastructure" as agent
substrate, not just GPU rental. The thing I keep coming back to: the unit of
deployment is shifting from container to agent, and the orchestration primitives
we have (k8s, Airflow) don't fit. New layer coming.
